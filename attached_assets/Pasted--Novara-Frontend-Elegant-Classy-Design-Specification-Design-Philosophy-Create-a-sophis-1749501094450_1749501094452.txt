# Novara Frontend - Elegant & Classy Design Specification

## Design Philosophy
Create a **sophisticated, elegant, and classy** React frontend for Novara following the aesthetic shown in the reference image. The design should be **minimalist, professional, and timeless** with a focus on typography, white space, and subtle design elements.

## Visual Design Requirements

### Color Palette
- **Primary**: Clean whites (#FFFFFF) and off-whites (#FAFAFA)
- **Secondary**: Soft grays (#F5F5F5, #E5E5E5) for backgrounds
- **Text**: Deep charcoal (#2C2C2C) and medium gray (#666666)
- **Accent**: Warm gold/bronze (#D4AF37) for highlights and CTAs
- **Borders**: Light gray (#E0E0E0) for subtle divisions

### Typography
- **Primary Font**: Elegant serif font (Playfair Display or similar)
- **Secondary Font**: Clean sans-serif (Inter or Poppins)
- **Headings**: Large, bold serif with generous letter spacing
- **Body Text**: Clean sans-serif, readable line height (1.6)
- **Font Sizes**: Generous scaling (48px+ for main headings)

### Layout Principles
- **Generous White Space**: Lots of breathing room between elements
- **Centered Content**: Max-width containers with centered alignment
- **Grid System**: Clean, organized grid layouts
- **Minimal Borders**: Subtle, thin lines when needed
- **Card-based Design**: Elegant cards with subtle shadows

## Component Design Specifications

### Buttons
```css
// Primary Button (Buy/Sell)
- Background: Linear gradient gold/bronze
- Text: White, medium weight
- Padding: 12px 32px
- Border radius: 8px
- Hover: Slight darkening, subtle lift
- Transition: Smooth 0.3s ease

// Secondary Button
- Background: Transparent
- Border: 2px solid gold/bronze
- Text: Gold/bronze color
- Hover: Fill with gold background
```

### Forms
```css
// Input Fields
- Border: 1px solid light gray
- Background: White
- Padding: 12px 16px
- Border radius: 6px
- Focus: Gold border, subtle shadow
- Labels: Elegant typography above inputs

// Form Containers
- Clean white cards with subtle shadows
- Generous padding and spacing
- Elegant headings
```

### Cards
```css
// Book/Blog Cards
- Background: White
- Border: 1px solid very light gray
- Shadow: Subtle, soft (0 2px 8px rgba(0,0,0,0.1))
- Border radius: 12px
- Padding: 20px
- Hover: Lift effect with increased shadow
- Transition: Smooth transform and shadow
```

## Responsive Design
- **Desktop**: Full layout with sidebar and grid
- **Tablet**: Adjusted grid columns, collapsible sidebar
- **Mobile**: Single column, mobile-friendly navigation

## Animations & Interactions
- **Hover Effects**: Subtle lifts, color changes
- **Page Transitions**: Smooth fade-ins
- **Loading States**: Elegant spinners
- **Micro-interactions**: Button press feedback
